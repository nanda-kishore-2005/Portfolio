# Platform Logos

Place the following logo images in this directory:

1. **leetcode.png** - LeetCode logo
   - Download from: https://assets.leetcode.com/static_assets/public/images/LeetCode_logo.png
   - Or search for "LeetCode logo" and download a PNG/SVG version

2. **codechef.png** - CodeChef logo
   - Download from: https://www.codechef.com/misc/fb-image-icon.png
   - Or search for "CodeChef logo" and download a PNG/SVG version

3. **codeforces.png** - Codeforces logo
   - Download from: https://codeforces.org/s/0/favicon-32x32.png
   - Or search for "Codeforces logo" and download a PNG/SVG version

4. **github.png** - GitHub logo
   - Download from: https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png
   - Or use the official GitHub logo from their brand assets

5. **linkedin.png** - LinkedIn logo
   - Download from: https://content.linkedin.com/content/dam/me/business/en-us/amp/brand-site/v2/bg/LI-Bug.svg.original.svg
   - Or search for "LinkedIn logo" and download a PNG/SVG version

**Recommended specifications:**
- Format: PNG or SVG (PNG preferred for consistency)
- Size: 48x48px to 128x128px (will be scaled to 48x48px in the component)
- Background: Transparent or white background
- Quality: High resolution for crisp display

**Note:** The component uses Next.js Image component which will automatically optimize the images. Make sure the file names match exactly as listed above.
